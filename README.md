# Spring AI + MCP + AI Agents Masterclass

This repository contains the source code for my Udemy course on building [**AI Agents**](https://www.udemy.com/course/ai-agents-java/) using Spring AI, Java, and the Model Context Protocol (MCP).
![](.doc/java-mcp.png)

⚡ Production-focused hands-on course for building real AI agent systems, not just chatbots.

---

## What you will learn

* Build AI Agents and Agentic Workflows using Spring AI and Java
* Design MCP-based systems using Tools, Resources, and Prompts
* Implement human-in-the-loop workflows using elicitation
* Handle long-running workflows with progress notifications
* Integrate OpenAI, Gemini, and local models (Ollama)
* Design production-grade tool architecture and error handling
* Build AI-powered Spring Boot microservices
* Apply real-world agentic design patterns and best practices
